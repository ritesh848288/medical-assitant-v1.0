# DoctorAI - Offline Medical Assistant Website

A complete multi-page medical consultation website powered by Mistral 7B LLM, running completely offline.

## Features

- 🔐 **User Authentication** - Register/Login system
- 💬 **AI Chat Consultation** - Talk to Mistral AI about medical concerns
- 🔍 **Symptom Checker** - Analyze symptoms and get preliminary guidance
- 📊 **User Dashboard** - View chat history and symptom checks
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 🔒 **Privacy First** - All data stored locally, no cloud dependency
- 🚀 **Mistral 7B** - Advanced offline AI model

## Pages

1. **Home** - Landing page with features and information
2. **About** - Information about the service
3. **Contact** - Contact form and information
4. **Login/Register** - User authentication
5. **Dashboard** - User's personal space with history
6. **Chat** - AI consultation interface
7. **Symptoms** - Symptom analysis tool
